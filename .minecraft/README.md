# MineBolso .minecraft

Copie suas versões, modpacks ou instalações exportadas para `versions/`.

Exemplos:

- `.minecraft/versions/Forge 1.20.1/`
- `.minecraft/versions/Fabric 1.21/`
- `.minecraft/versions/Better MC/`

O MineBolso detecta automaticamente loaders, mods, arquivos de inicialização e problemas de integridade.
